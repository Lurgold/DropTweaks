DropTweaks est un script qui permet que quand on casse du verre ou des blibliotèques ca les drop dirrectement

Le script au cas ou :)

on break of glass:
	drop glass

on break of thin glass:
	drop thin glass

on break of bookshelf:
	cancel event
	set block to air
	drop bookshelf
  
  
  Aucun crédit nécessaire !
